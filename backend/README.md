# MentorLink Node.js backend with MariaDB server
Gunakan Node.js untuk menjalankan server dengan:
```bash
node server.js
```

Untuk database, gunakan MySQL atau MariaDB dan buatlah user bernama "mentorlink_user" dengan nama database "mentorlink". Untuk password yang akan diberi bebas, silahkan mengubah isi `db.js` terlebih dahulu.
